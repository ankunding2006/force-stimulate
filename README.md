# 力的合成可视化仿真

这是一个使用 Python 和 Pygame 开发的交互式力的合成可视化仿真程序。

## 功能特性

### ? 核心功能

-   **交互式力向量创建**: 通过鼠标拖拽创建力向量
-   **实时合力计算**: 自动计算并显示所有力的合力
-   **可视化展示**: 直观显示各个力向量和合力
-   **动态编辑**: 拖拽修改力向量的大小和方向

### ? 交互操作

-   **左键拖拽**: 创建新的力向量或移动现有力向量的端点
-   **右键点击**: 删除指定的力向量
-   **键盘操作**:
    -   `C键`: 清空所有力向量
    -   `G键`: 切换网格显示
    -   `H键`: 切换力分量显示

### ? 可视化元素

-   **力向量**: 不同颜色的箭头表示不同的力
-   **合力**: 紫色粗箭头从屏幕中心显示合力
-   **数值标签**: 显示每个力的大小和角度
-   **坐标系**: 标准的 XY 坐标轴
-   **网格**: 可选的参考网格
-   **分量显示**: 显示力在 X、Y 方向的分量

### ? 信息面板

-   实时显示力的数量
-   显示合力的大小和方向
-   显示选中力向量的详细信息
-   操作说明帮助

## 安装和运行

### 环境要求

-   Python 3.7+
-   pygame 2.5.2+
-   numpy 1.24.3+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python force_simulator.py
```

## 使用方法

1. **创建力向量**: 在空白区域按住左键拖拽，释放鼠标创建力向量
2. **修改力向量**: 点击力向量选中，然后拖拽端点修改大小和方向
3. **删除力向量**: 右键点击要删除的力向量
4. **观察合力**: 紫色箭头显示所有力的合力，从屏幕中心开始
5. **查看信息**: 左上角面板显示详细的数值信息

## 物理原理

程序基于向量加法原理计算力的合成：

-   合力 = Σ(各分力的向量和)
-   力的大小 = √(Fx? + Fy?)
-   力的方向 = arctan(Fy/Fx)

## 技术实现

### 核心类

-   `Vector2D`: 二维向量数学运算
-   `Force`: 力向量的表示和渲染
-   `ForceSimulator`: 主仿真类，处理交互和绘制
